When the PP site was re-launched it required a tool to enable easy navigation of the 80 or so items in its library. I came-up with a custom 3 panel browser enabling users to drill-down via categories to select what they wanted. Initially this was based on jquery and all the items were hard-coded into the html with the image thumbnail src attributes generated on the fly so as to reduce http requests and page load.
